The Clone Wars API

The Clone Wars API is a HTTP web service for data about the animated series Star Wars: The Clone Wars. You can find information about the characters, planets, species, and vehicles that have appeared in series.
You can try out the API using this link. Try it out! There, you can use a demo of the service. It will show you the data's layout and their request URLs.
