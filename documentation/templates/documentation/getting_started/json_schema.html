JSON Schema

This API only supports JSON format. All responses will either be a single object or an array of JSON objects. Every object will have basic fields: id, name, and link.
Most objects will have additional details under the info key. The documentation will go more in-depth in their respective 'Resource' section.
Example: 
{
    "id": 12,
    "name": "Ahsoka Tano",
    "link": "https://clonewars.fandom.com/wiki/Ahsoka_Tano",
    "info": {
        "gender": "Female",
        "species": "Togruta",
        "homeworld": "Shili",
        "affiliation": [
            "Jedi Order (formerly)",
            "Galactic Republic (formerly)",
            "Nite Owls"
        ],
        "occupation_rank": [
            "Jedi Padawan (formerly)",
            "Jedi Commander (formerly)"
        ]
    }
}
