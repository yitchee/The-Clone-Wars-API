Authentication

In order to use this API, you will need to include your API key with every request under the header X-API-KEY. If you do not have an API key, you can sign up for one here.
Below is an example of how to access the API using JavaScript's fetch(): 
fetch("https://api.clonewarsapi.com/characters/random/",
    {
        method: 'GET',
        headers: {
            'X-API-KEY': '<<API_KEY>>',
    }).then(data => {...});
